CASE-001: SIEM Threat Detection Lab

Completed: 2024

Deployed and configured Splunk Enterprise SIEM platform to analyze 68 million+ events from the BOTSv2 (Boss of the SOC v2) dataset. Built custom dashboards and SPL queries to identify malicious IPs, command-and-control communications, and lateral movement patterns.

Objectives

  • Set up Splunk Enterprise in a lab environment
  • Index and parse 68M+ security events
  • Create detection rules using SPL (Search Processing Language)
  • Build operational dashboards for threat hunting
  • Generate incident reports with geospatial analysis

Key Findings

  • Identified 12 malicious IP addresses conducting data exfiltration
  • Detected C2 (Command & Control) beaconing patterns
  • Discovered privilege escalation attempts on 5 critical servers
  • Mapped attack timeline across multiple systems

Tools & Technologies

Splunk Enterprise SPL BOTSv2 Dataset Data Visualization Threat Intelligence

Outcomes

  • Mastered SPL query syntax for complex security investigations
  • Learned SIEM best practices for alert tuning and reduction
  • Built proficiency in dashboard design for SOC operations
  • Developed skills in threat attribution and hunting

CASE-003: Network Forensics & Packet Analysis

Completed: 2024

Conducted deep-dive network forensics analysis using Wireshark on captured PCAP files. Performed TCP stream reconstruction, protocol analysis, and anomaly detection through baseline comparison techniques on real-world network traffic.

Objectives

  • Master Wireshark packet capture and filtering
  • Analyze TCP/IP protocol behavior
  • Reconstruct TCP streams for conversation analysis
  • Identify protocol anomalies and suspicious patterns
  • Establish network baselines for comparison

Analysis Performed

  • Identified 8 suspicious outbound connections to known C2 servers
  • Detected DNS tunneling attempts for data exfiltration
  • Found evidence of port scanning and enumeration activity
  • Reconstructed HTTP session data showing credential theft
  • Analyzed SSL/TLS certificate anomalies indicating potential MITM attacks

Tools & Technologies

Wireshark tcpdump PCAP Analysis Protocol Analysis Zeek IDS

Outcomes

  • Developed expertise in packet-level threat hunting
  • Learned to identify encrypted vs. unencrypted malicious traffic
  • Built skills in protocol-specific anomaly detection
  • Gained understanding of network-based indicators of compromise

CASE-004: Honeypot Deployment & Threat Intelligence

Completed: 2024

Designed and deployed isolated honeypot networks on Kali Linux to passively observe attacker behavior. Collected TTPs (Tactics, Techniques, Procedures), attack patterns, and developed threat profiles from real-world exploit attempts.

Objectives

  • Set up honeypot VMs with intentional vulnerabilities
  • Deploy network segmentation and monitoring
  • Capture attacker behavior and exploitation techniques
  • Document TTP patterns using MITRE ATT&CK framework
  • Extract indicators of compromise (IOCs)

Honeypot Architecture

  • Isolated network segment (DMZ simulation)
  • Vulnerable web application server (Damn Vulnerable Web Application)
  • SSH service with weak credentials
  • File sharing services with default passwords
  • Comprehensive logging and traffic capture

Tools & Technologies

Kali Linux VirtualBox DVWA Firewall Rules MITRE ATT&CK

Outcomes

  • Observed 47 successful attack attempts over 2-week period
  • Documented attacker TTPs mapped to MITRE framework
  • Extracted 156 IOCs (IPs, domains, file hashes)
  • Built threat intelligence reports for security team
  • Developed understanding of real-world attack methodologies
